# Punk Brew

<img align="right" width="120" src="img/sample-icon.png">

Android application which represents [BrewDog](https://www.brewdog.com) beer catalogue receiving data via Punk API and store data locally. In this application you can view the entire catalog of BrewDog kinds of beer, detailed information about each kind and add some items to favorites.
